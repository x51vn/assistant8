Core commands for this repo:\n- `npm run build` - production build via Vite\n- `npm run build:watch` - watch build\n- `npm run test:unit` / `npm run test:unit:run` - Vitest\n- `npm run test:e2e` / `npm run test:e2e:smoke` - Playwright extension tests\n- `npm run check:repo-hygiene` - repository hygiene guard\n- `npm run check:openspec` - validate OpenSpec artifacts\n- `npm run smoke:extension` - build plus extension smoke check\n- `openspec status --change \"<name>\" --json` - inspect artifact status\n- `openspec instructions <artifact> --change \"<name>\" --json` - fetch artifact template/instructions\n- `openspec validate <change> --strict --no-interactive` - strict validation for a change
