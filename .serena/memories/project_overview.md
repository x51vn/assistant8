Assistant8 is a Chrome MV3 extension (repo name chatgpt-assistant) focused on portfolio/trading assistance, ChatGPT interaction/history capture, writing/research helpers, and Supabase-backed user workflows. Stack: JavaScript/ES modules, Preact, Vite, Vitest, Playwright, Supabase JS. High-level structure: `src/background` service worker and handlers, `src/content.js` plus `src/content/*` content automation, `src/ui-preact` UI, `src/shared` contracts/validators/mappers, `src/platform` browser abstractions, `tests/unit|integration|e2e`, `openspec` for spec-driven changes. Key architecture constraints from repo docs: MV3 service worker must stay stateless, listeners register synchronously at top level, durable business data belongs in Supabase with RLS, realtime should live in UI not background, and message contracts must use shared schema/helpers.
