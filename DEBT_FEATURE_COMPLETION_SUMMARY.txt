HOW THE FEATURE WORKS
================================================================================

Net Worth Calculation Formula:
  Net Worth = Total Assets + Stocks - Total Debts

Example:
  Assets:    100,000,000 VND
  Stocks:     20,000,000 VND
  Debts:     (30,000,000 VND)
  ─────────────────────────────
  Net Worth:  90,000,000 VND ✓

User Flow:
  1. Click "Thêm tài sản" (Add Asset)
  2. Select "Khoản vay" (Debt)
  3. Enter: Name, Amount, Lender, Interest Rate, Due Date
  4. Save → Appears in list with RED tag
  5. Net Worth Summary shows debt in RED stat
  6. Total = Assets - Debt (automatic)

UI Visual Cues:
  - Red "Khoản vay" tag on debt items
  - Minus sign prefix on debt amounts
  - Red debt stat in summary (when > 0)
  - Red color: #dc3545 (error/liability theme)
