# GPT-001-002 Add .env.example with Supabase placeholders

## Project Context (MUST READ)
Project sẽ chuyển sang Supabase. Developers cần .env.example để biết Supabase config keys cần thiết. Audit chỉ ra không có .env.example.

## Parent Ticket
GPT-001 (Baseline audit & Architecture-Code mapping)

## Timebox
15-30 phút.

## Goal
Tạo .env.example với Supabase placeholders.

## Inputs
- docs/ARCHITECTURE.md (Supabase config section)

## Requirements
1. Create .env.example:
   ```env
   # Supabase Configuration
   VITE_SUPABASE_URL=https://your-project.supabase.co
   VITE_SUPABASE_ANON_KEY=your-anon-key-here
   
   # Optional: Environment
   VITE_ENV=development
   ```
2. Add .env to .gitignore nếu chưa có.
3. Update README.md với instructions copy .env.example → .env.
