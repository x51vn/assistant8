🚀 HOW TO USE (3 SIMPLE STEPS)
═════════════════════════════════════════════════════════════════════════════════

STEP 1: Install Extension (2 minutes)
────────────────────────────────────
  1. Open Chrome
  2. Go to: chrome://extensions/
  3. Turn ON "Developer mode" (top right)
  4. Click "Load unpacked"
  5. Select: d:\dtx8\chatgpt-extension
  6. ✓ Done! Icon appears in toolbar

STEP 2: Configure (1 minute)
────────────────────────────
  1. Click extension icon
  2. Go to "Cấu hình" (Settings) tab
  3. Type your prompt: "Hãy giải thích AI là gì"
  4. Click "Lưu cấu hình" (Save)

STEP 3: Run & See Results (30 seconds)
──────────────────────────────────────
  1. Go to "Kết quả" (Results) tab
  2. Click "Chạy ngay" (Run now)
  3. Wait 10 seconds
  4. ✓ Results appear!

❓ QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

Problem: "Cannot get results"
  → Check ChatGPT tab is open
  → Reload extension (chrome://extensions > Reload)
  → Make sure logged in to ChatGPT

Problem: "Prompt not sent"
  → Check prompt is not empty
  → Wait for ChatGPT to load completely
  → Try again after 3-5 seconds

Problem: "Extension won't open ChatGPT"
  → Check permissions: chrome://extensions > Details > Site access
  → Allow on chatgpt.com site

Problem: Other issues
  → See INSTALL.md "Troubleshooting" section
  → Open DevTools (F12) to check logs
  → Run test-helper.js on ChatGPT tab
