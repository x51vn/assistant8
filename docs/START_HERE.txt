❓ CẦN GIÚP?
═════════════════════════════════════════════════════════════════════════════════

Lỗi: "Không lấy được kết quả"
  → Kiểm tra ChatGPT tab mở không
  → Reload extension: chrome://extensions → Reload
  → Đảm bảo đã đăng nhập ChatGPT

Lỗi: Prompt không được gửi
  → Kiểm tra prompt không trống
  → Chờ ChatGPT tab load xong
  → Thử lại sau vài giây

Extension không mở ChatGPT
  → Kiểm tra permission: chrome://extensions > Details > Site access
  → Chọn "Allow on site"

Khác:
  → Xem phần Troubleshooting trong INSTALL.md
  → Mở DevTools (F12) xem console logs
  → Chạy test-helper.js trên ChatGPT tab

💡 TIPS & TRICKS
═════════════════════════════════════════════════════════════════════════════════

Tip 1: Viết Prompt Tốt
  ✓ Ngắn gọn (1-2 câu)
  ✓ Rõ ràng (chỉ rõ yêu cầu)
  ✓ Cụ thể (tránh mơ hồ)

Tip 2: Giữ ChatGPT Tab Mở
  ✓ Để tab ChatGPT luôn mở
  ✓ Nếu đóng, extension sẽ mở lại
  ✓ Không reload trang ChatGPT khi chạy

Tip 3: Thời Gian Chạy
  ✓ Minimum 1 phút (quá nhanh sẽ lỗi)
  ✓ Recommend 5-10 phút
  ✓ Maximum 24 giờ (1440 phút)

Tip 4: Đăng Nhập ChatGPT
  ✓ Đảm bảo đã đăng nhập
  ✓ Giữ session sống
  ✓ Không logout trong lúc extension chạy
